# Claude Code 接入

## 建议接入方式

- 使用 `adapters/claude/.claude/skills/` 作为本地 skill pack
- 直接使用本仓库中的 `skills/shared/` 作为 Claude skill 内容
- 按需安装官方 skills

## 最小安装

1. 复制 `adapters/claude/.claude/skills/` 到目标项目
2. 阅读 `start-agent-team` 和 `create-agent`
3. 按项目需要启用角色

## 角色建议

- Team Lead
- Product Manager
- Tech Lead
- Engineer
- QA Engineer
- Researcher
- Platform/SRE
