# Dependencies

## Required Baseline

- `git`
- text search tooling such as `rg`
- issue tracker
- pull request workflow

## Recommended Tooling

- GitHub CLI
- a stacked PR tool such as `git-stack` or an equivalent `gstack` workflow
- CI automation

## Recommended Reuse

- Anthropic official skills for planning, architecture, review, and verification
- Superpowers as a reference for cross-platform packaging and install patterns
