# Claude Code 接入

## 建议接入方式

- 使用 `adapters/claude/.claude/skills/` 作为本地技能包
- 直接使用本仓库中的 `skills/shared/` 作为 Claude 技能内容
- 按需安装官方技能

## 最小安装

1. 复制 `adapters/claude/.claude/skills/` 到目标项目
2. 阅读 `start-agent-team` 和 `create-agent`
3. 按项目需要启用角色

## 角色建议

- 团队负责人
- 产品经理
- 技术负责人
- Engineer
- 测试工程师
- Researcher
- 平台 / SRE
