# Issue Naming Convention

## Issue Types

| Type | Description |
|------|-------------|
| feature | 功能需求 |
| bug | 缺陷修复 |
| discuss | 讨论 |

## Priority Levels

| Priority | Description |
|----------|-------------|
| P0 | 最高 |
| P1 | 高 |
| P2 | 中 |
| P3 | 低 |

## Naming Format

```
Issue_${type}#${id}:${name}
```

### Examples

- `Issue_feature#1:添加Telegram Bot部署说明`
- `Issue_bug#2:修复CI跳过逻辑`
- `Issue_discuss#3:讨论CI策略调整`
