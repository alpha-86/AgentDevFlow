# AgentDevPipeline Team Setup

1. Create the team.
2. Create agents through `create-agent`.
3. Start work through `start-agent-team`.
4. Load shared roles and workflows from the repository.

## Required Shared Assets

- `skills/shared/agents/`
- `skills/shared/workflows/`
- `skills/shared/templates/`
- `prompts/zh-cn/`

## Rule

All required role and workflow content must be loaded from this repository only.
