# 日会 Workflow

## 目标

同步状态、阻塞、优先级和下一步动作。

## 输入

- 当前 todos
- 打开的 reviews
- 当前交付阶段
- 进行中的关联 issues

## 议程

1. 昨日进展
2. 今日计划
3. blockers
4. gate 或 review 风险
5. todo 跟进
6. 当日需要决策的事项

## 输出

- 日会 memo
- 更新后的 owner 和 due date
- 必要的升级动作

## 日会 Memo 必填字段

- 关联 issue IDs
- 每个 issue 当前 gate
- blocker owner 和升级路径
- 下一次 review 或 gate 准入条件
